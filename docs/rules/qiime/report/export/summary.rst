Qiime - Tools Export
------------------------

Exports any qiime visualization (qzv) to HTML.

**Location**

- *Filepath:* <SnakeLines_dir>/rules/qiime/report/export/export.snake
- *Rule name:* qiime__tools_export

**Input(s):**

- *qzv:* qiime visualization

**Output(s):**

- *html:* index.html of exported visualization
